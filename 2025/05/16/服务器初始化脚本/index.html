    服务器初始化脚本

    一些服务器初始化/优化脚本

    前情摘要：最近购置了几台服务器 按照我个人的习惯 必然是要omz+p10k安排上的 但是一次两次还好 服务器多了 配起来就很累了 所以我搓了个小脚本 用来自动化这一过程 效果如图
    当然大部分效果都是花里胡哨的 别问为什么lsd要分开来装 问就是自己用的时候才发现服务器是debian11的 没有lsd 气笑了

    一键运行命令

    wget -qO- https://raw.githubusercontent.com/swim233/shell_scripts/main/init.sh | bash

    该脚本会安装一些常用软件如 btop git fzf neovim 等 会给zsh装上 syntax-highlight auto-suggestion extract等插件 还有一个花里胡哨的登录ascii

    这个ascii也有独立的脚本 你可以在这里找到他并自行放到你的~/.zshrc里 需要注意的是 如果同时在使用p10k的话 在~/.zhsrc添加这类有输出的内容的话 需要将其放在配置文件最顶部 否则会报错

    另外附上群友提供的优化脚本 有需要的也可以使用试试

    wget -qO- https://raw.githubusercontent.com/swim233/shell_scripts/main/optimize.sh | bash
